CSS là gì?

CSS (Cascading Style Sheets) là ngôn ngữ được sử dụng để tìm và định dạng lại các phần tử được tạo ra từ HTML. Nói một cách dễ hiểu, CSS là ngôn ngữ dùng để tạo ra phong cách của website. Trong khi HTML đóng vai trò định dạng phần tử trên website như tạo đoạn văn bản, tiêu đề, bảng,... CSS sẽ giúp thêm style vào các phần tử HTML đó. Các style cụ thể thường được thêm là đổi bố cục, màu sắc trang, font chữ, màu chữ, cấu trúc mới,... Ngôn ngữ đánh dấu phổ biến nhất là HTML nhưng bạn có thể bắt gặp thêm một số ngôn ngữ khác như SVG, XML,...

Docker là gì?

Docker cho phép lập trình viên đóng gói ứng dụng thành Container, là các thành phần thực thi được chuẩn hóa kết hợp mã nguồn của ứng dụng cùng với các thư viện của hệ điều hành và các thư viện cần thiết khác để thực thi đoạn mã trong mọi môi trường. Container có tác dụng làm đơn giản hóa việc phân phối các ứng dụng phân tán và ngày càng trở nên phổ biến khi các tổ chức chuyển sang phương pháp Cloud Native, môi trường Hybrid-cloud và Multi Cloud.
